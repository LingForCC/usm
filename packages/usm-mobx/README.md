# usm-mobx

Using `usm-mobx` to develop a modular application system. **`usm-mobx` can help you make more concise and intuitive OOP when using some state library**, it can directly support MobX.

## Install

Use yarn:

```bash
yarn add usm-mobx
```

Or use npm:

```bash
npm install --save usm-mobx
```
