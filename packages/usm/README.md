# usm

Using `usm` to develop a modular application system. **`usm` can help you make more concise and intuitive OOP when using some state library**, it can directly support Native Module/Angular.

## Install

using yarn:

```bash
yarn add usm
```

Or npm:

```bash
npm install --save usm
```
